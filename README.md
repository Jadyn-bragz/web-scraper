# web-scraper
Tech stack:
This project is developed using React.js for UI interface ,Node.js and puppeteer for the web scraping logic, and OAuth 2 along with express js for authentication.

Prior to running:
1. Ensure there is no process running on port 5000. If there is ,
   a. use netstat -ano | findstr :5000 to find the pid 
   b. kill the process using taskkill /pid pidno /F

Steps to run: 
1.Run this installation script to ensure you have the necessary libraries installed.

2.Run the command node index.js
